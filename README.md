# ML Lab

This repository documents my journey into Machine Learning, with a focus on building models from first principles and understanding the underlying concepts rather than just applying libraries.

## Motivation

The goal of this project is to develop a deep and practical understanding of Machine Learning by combining:

- Theoretical insights (from physics and mathematics)
- Hands-on implementation
- Clean and reproducible project structure

This repository serves both as a learning diary and as a foundation for future, more advanced ML projects.

---
